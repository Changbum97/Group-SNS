# SNS Shopping Mall 프로젝트

## 배포 사이트

http://ec2-52-79-82-151.ap-northeast-2.compute.amazonaws.com:8085/

## 개발 환경

- Language: JAVA 11
- Framework: SpringBoot 2.7.11
- Build: Gradle 7.6.1
- Server: AWS EC2
- Deploy: Docker
- CI/CD: Github Actions
- Database: MySQL 8.0
- File Storage: AWS S3

## 기능

- 결제
- 1대1 채팅
- Redis
- 이메일 인증, 전화번호 인증
- 알람
- Jwt Token
- Oauth
- 에러페이지
- 관리자 기능
- Controller, RestController 분리
